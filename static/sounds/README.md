# Sound Assets

## Notification Sound

The notification sound (`notification.mp3`) is used when the chat widget opens.

### Usage
The sound is played when a user clicks to open the chat widget, respecting browser autoplay policies.

### Replacement Instructions
To replace the notification sound:
1. Download your preferred sound effect in MP3 format
2. Name it `notification.mp3`
3. Place it in this directory (`/static/sounds/`)
4. Update this README with the new attribution if required
